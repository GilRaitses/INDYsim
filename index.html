Introduction

Introduction

Interface Dynamics Framework

Interface Dynamics Simulation (INDYsim) investigates behavior as the interface between perception and adaptation, treating behavioral transitions as both signal and medium. This framework bridges computational neuroscience with media theory, modeling how sensory representation translates into neural activity and ultimately manifests as observable behavior.

Project Overview

This term project develops a stimulus-response modeling framework for Drosophila larval behavior using event-hazard methods from survival analysis and generalized linear models. The project uses trajectory data from mechanosensation experiments to estimate time-varying hazard rates for behavioral events conditioned on LED stimulus intensity, temporal history, and contextual features.

Problem Statement

Problem Statement

Research Question

How can larval behavioral responses to time-varying LED stimuli be modeled using event-hazard methods to predict turn initiation rates, behavioral state transitions, and trajectory-level statistics?

Modeling Challenge

Behavioral events occur stochastically with rates that depend on stimulus features (intensity, temporal history, pulse timing), contextual features (speed, orientation, wall distance), and individual heterogeneity. This project develops a structured hazard model that captures temporal dependencies and supports trajectory simulation.

Methodology

Proposed Methodology

Core Model: Stimulus-Locked Event-Hazard GLM

For each behavioral event type E ∈ {turn, stop, reverse}, define the time-varying hazard rate:

λE(t) = exp{β0,E + φET[s ⋆ κ](t) + x(t)TβE}

Temporal Kernel Design

The kernel κ(τ) captures latency effects (peak response at delay τ0 ≈ 0.5-2 seconds), adaptation (decay over longer delays), and anticipation. Using raised cosine basis functions with knots spanning [-2, 20] seconds relative to stimulus onset.

Conclusion

Conclusion

This project applies simulation modeling methods from ECS630 to biological behavioral data, developing a stimulus-response model that simulates larval trajectories under different experimental conditions. The event-hazard framework models stochastic behavioral events, while the DOE methodology explores stimulus parameter space systematically. Results use Arena-style statistics including confidence intervals and across-replications summaries, showing how simulation principles extend from manufacturing systems to biological processes.
