\section{Data Availability}

All code, data, and results from this study are available at the public GitHub repository: \url{https://github.com/GilRaitses/InDySim}

The \texttt{phenotyping\_followup/} directory contains the complete analysis pipeline organized as follows:

\texttt{code/} contains 16 numbered analysis scripts (01--16) covering the full pipeline from initial clustering through identifiability analysis, plus utility scripts for cross-validation, data generation, and figure generation. A README describes the purpose of each script.

\texttt{data/simulated\_phenotyping/} contains 300 simulated trajectories used for round-trip validation.

\texttt{results/} contains intermediate analysis outputs including kernel fits, clustering results, hierarchical Bayesian posteriors, and validation metrics.

\texttt{figures/} contains all manuscript figures in PDF and PNG formats.

\texttt{presentation/} contains Beamer slides with FAQ annotations for each key figure.
