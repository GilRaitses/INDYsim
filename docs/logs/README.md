# Daily Logs

This folder contains daily logs documenting work progress on the INDYsim project.

## Structure

```
docs/logs/
├── README.md (this file)
├── YYYY-MM-DD.md (daily logs)
└── YYYY-MM-DD/ (subfolders for specific work items)
    └── [work-specific documentation]
```

## Daily Log Format

Each daily log (`YYYY-MM-DD.md`) should include:
- Objective for the day
- Active agents and their tasks
- Progress made
- Blockers encountered
- Next steps

## Work Subfolders

For significant work items, create a subfolder `YYYY-MM-DD/work-item-name/` containing:
- Detailed documentation
- Handoff documents
- Analysis notes
- Related files

## Example

```
docs/logs/
├── 2025-11-10.md (daily log)
└── 2025-11-10/
    ├── REPOSITORY_ORGANIZATION.md
    ├── PATH_CLEANUP_NOTES.md
    └── SUMMARY.md
```
