### 2. Rate Calibration

**Problem**: Model underproduced events by ~40%
- Simulated: 830 events vs Empirical: 1407 events
- Rate ratio: 0.59 (target: 0.8-1.25)

**Solution**: Calibrated intercept
```
original_intercept = -6.76
calibration_factor = 1407 / 830 = 1.695
calibrated_intercept = -6.76 + log(1.695) = -6.23
```
